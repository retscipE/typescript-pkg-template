# Typescript NPM Package Template
## Make sure to do:
```js
npm install
```
## Before you start to do anything!
### To test your package, do:
```js
npm test
```

All tests should be written like the one in **Greeter.test.ts**

### Finally, to publish your package do:
```
npm adduser or npm login
```
### And follow the steps there to login to your npm account, then do:
```
npm publish
```
### And then your package is live on npmjs.com/package/[your package name]
